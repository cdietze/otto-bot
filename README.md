# otto-bot
A bot implemenation for [bots](https://github.com/markusfisch/bots).

To run call

    ./gradlew --console=plain execute -Pmode=MODE
    
where `MODE` is one of

- `Hello`
- `Escape`
- `Collect`
- `Word`
- `Avoid`
